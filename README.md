## Descrição do Projeto
Este projeto foi desenvolvido como parte de um desafio prático da DIO, com o objetivo de simular a implementação de serviços da Amazon Web Services (AWS) em uma farmácia fictícia, visando a redução imediata de custos operacionais e a modernização da infraestrutura de TI.
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon EC2
- Foco: Computação em nuvem sob demanda
- Caso de uso: Utilizado para hospedar o sistema de gestão da farmácia (vendas, estoque e cadastro de clientes).
Substitui servidores físicos locais, permitindo escalabilidade, pagamento por uso e redução de custos com hardware e energia.

Etapa 2: 
- Amazon S3
- Foco: Armazenamento escalável e de baixo custo
- Caso de uso: Armazenamento de documentos fiscais, relatórios financeiros, backups do sistema e imagens de produtos.
Aplicação de políticas de classes de armazenamento para reduzir custos com dados pouco acessados.

Etapa 3: 
- AWS Cost Explorer
- Foco: Monitoramento e otimização de custos
- Caso de uso: Utilizado para acompanhar os gastos da infraestrutura AWS, identificar recursos ociosos e auxiliar na tomada de decisões para redução contínua de custos.



## ConclusÃ£o
A implementação dos serviços AWS demonstrou ser uma solução eficiente para empresas do setor farmacêutico que buscam reduzir custos e aumentar a eficiência operacional. A adoção da nuvem possibilita maior flexibilidade, controle financeiro e preparo para crescimento futuro.
